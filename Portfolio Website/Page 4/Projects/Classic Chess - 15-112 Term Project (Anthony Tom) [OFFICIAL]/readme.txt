Project Title: Classic Chess

Project Description: I created a traditional chess game with a wide variety of features using cmu_graphics. The program contains ChessBoard and Piece classes, In addition, the program contains methods that check if moves are legal and move pieces in the 2D list chess board. There are two ways of playing: Single Player, where a player can play against an AI, and Multiplayer, where two players can play against each other. Players click or drag/drop two cells on the chess board to select and move pieces. Players can select a king and then a rook to castle (if valid). Players can select a pawn in the top row and press a key in ‘bnqr’ to promote it. The possible legal moves of clicked pieces are displayed. The user can select different AI difficulty modes. The Easy AI moves random pieces using the Python random module. The Medium AI moves pieces using a modified version of the minimax algorithm (by Anthony Tom). The Hard AI immediately captures random pieces using the Python random module. The Extreme AI immediately captures the opponent’s capturable piece with the highest score. Additional Features:  In Player vs. Player mode, alternating chess timers are implemented. There are information screens (hold ‘i’) and game moves screens (hold ‘m’ after the game has started). Chess piece images are presented, select cells are highlighted blue, legal move cells are highlighted green, and illegal move cells are highlighted red. Sound effects are made after legal and illegal moves are made.

Run Instructions: The following folders are required to run the program: chess_images (12 images), sound_effects (2 sounds), and src (chess.py, chess_classes.py, minimax_program.py, cmu_graphics version 1.1.39 module folder from Carnegie Mellon University). The additional modules needed are cmu_graphics (which should be placed in the src folder), and the Python built-in modules copy, random, time. The chess.py program must be run to display the chess program.

Shortcut Commands: At the beginning of the game, the user can press numbered keys 0-4 to select the game mode. The user can press key ‘s’ to restart the game (go back to the start screen). The user can press ‘e’ repeatedly to carry out automated computer moves (for both black and white). Arrow keys (up, down, left, right) can be pressed to move selected cells on the board. The user can hold key ‘i’ to view the information screen. The user can hold key ‘m’ after the game has started to view the game moves screen. The user can also press key ‘c’ to click pieces and key ‘d’ to drag and drop pieces. The user can click a pawn in the top row and press ’b’, ‘n’, ‘q’, or ‘r’ to promote it.
